A modern, responsive portfolio website showcasing my skills as a **Full-Stack Developer**, **DevOps Engineer**, and **Game Developer**. Built with modern web technologies and featuring smooth animations, interactive elements, and a clean design.

## 🌟 Features

- **Responsive Design** - Optimized for all devices (desktop, tablet, mobile)
- **Modern UI/UX** - Clean, professional design with smooth animations
- **Interactive Portfolio** - Filterable project showcase by category
- **Dynamic Content** - Animated text headers and scroll effects
- **Contact Form** - Functional contact form with PHP backend
- **Blog Section** - Integrated blog for sharing insights
- **Performance Optimized** - Fast loading times and optimized assets
- **Cross-browser Compatible** - Works seamlessly across all modern browsers

## 🛠️ Technologies Used

### Frontend
- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with animations
- **JavaScript** - Interactive functionality
- **Bootstrap 4** - Responsive grid system

- **Alpine.js** - Lightweight JavaScript framework
- **jQuery** - DOM manipulation and animations

- **Owl Carousel** - Touch-enabled slider

- **Animate.css** - CSS animation library

### Backend
- **PHP** - Contact form processing

## 🚀 Quick Start

### Prerequisites
- A modern web browser
- Local web server (optional for development)
- Python 3.x (for local server)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rahuljoshi07/Portfolio-Website.git
   cd Portfolio-Website
   ```

2. **Start a local server**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```

## 🔧 Customization

### Colors
The website uses a dark theme with yellow accent colors. To customize:
- Edit `assets/css/main.css` for primary styling
- Modify TailwindCSS classes in HTML for quick changes

### Content
- Update `index.html` for main content
- Replace images in `assets/img/` directory
- Modify project data in the portfolio section

### Contact Form
- Configure `contact.php` with your email settings
- Update SMTP settings if needed
- Modify form fields as required
